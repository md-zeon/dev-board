Question-1: What Are The Different Ways To Select An Element In The DOM?

There are several ways to select elements in the DOM using JavaScript:
We can use document.getElementById('id') to select a single element with a specific ID.
We can use document.getElementsByClassName('class') to get all elements with a specific class. it returns a html collection
We can also Use document.getElementsByTagName('tag') to select all elements with a specific tag name.
We use document.querySelector('selector') to select the first element that matches a CSS selector
We use document.querySelectorAll('selector') to select all elements that match a CSS selector. it returns a NodeList.

Question-2: What Is The Difference Between innerHTML, innerText And textContent?

if you use innerHTML, it can include any HTML tags like div or span. It can also get or set the HTML content inside an element.
if you use innerText, it can get or set the text inside an element, but it can't do anything with the text that's not visible.
On the other hand, textContent gets or sets all the text inside an element, including hidden text and spacing. it can change text that's not visible. It doesn't care about styling or visibility.

Question-3: What Is Event Delegation In The DOM?

Event delegation is a technique where you add a single event listener to a parent element instead of adding listeners to multiple child elements. This works because events "bubble up" from the child to the parent.

Question-4: What Is Event Bubbling In The DOM?

Event bubbling is when an event triggered on a child element. it "bubbles up" through its parent elements in the DOM tree.
What I mean is If you click a button inside a div, the click event will first trigger on the button, then on the div, and then on any other parent elements all the way up to the document.

Question-5: How Do You Create, Add and Remove Elements Using JavaScript?

I Use document.createElement('tag') to create a new element, and use parentElement.appendChild(newElement) to add the new element and use parentElement.removeChild(childElement) to remove an element.
